# XiDunKu 西墩庫

xidunku 收集建立西大墩附近數位領域資源並發展相關教育訓練素材。

[XiDunKu 西墩庫 - Github](https://dltdojo.github.io/xidunku/)

西墩庫分成三個部份

- inku 入庫
- keepku 守庫
- outku 出庫

虛擬西墩庫區

- 實體映射
  - 台中市中央公園周圍1000m
  - 台中市公車路線周圍100m
- 虛擬服務
