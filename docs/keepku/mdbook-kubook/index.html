mdbook-kubook

prerequisites

  • java
  • plantum.jar

install mdbook-kubook

$ git clone https://github.com/dltdojo/xidunku.git && cd xidunku\keepku\mdbook-kubook
$ cargo install --path .

book.toml

[preprocessor.kubook]
command = "mdbook-kubook"
renderer = ["html"]
java-path="/usr/bin/java"
plantuml-jar="/usr/local/share/plantuml.jar"
